Mechanim Animation From Transform

17-10-2016

Gamedev / Helpers
#software #unity3d

The below example demonstrates how to implement mechanim animations on objects based on their transform (movement) each frame.

This means that it can be applied to both player controlled objects and objects being manipulated via code/automagically.

Example

An example of this can be seen in the following video (video is from a time travel prototype game that I am developing).



Code

(Scroll down to see small explanation after code)

Each update the Velocity (Vector3) of the object is taken (this is the difference between this frames transform and the previous frames transform).

This value is then passed to the animator (if one exists) which sets the specified movement float (“Forward“ in my example)

Reusing this code

If using this code on your own projects you should assume 1 for the variables T.TimeScaleAbs and T.TimeScale.
